# Audio Annotation Tool  
（音声アノテーション支援ツール）

本ツールは、**機械学習（音声分類・回帰タスク）向けの学習データセット作成を支援するデスクトップアプリケーション**です。  
GUI上で音声を再生しながら直感的にラベリングを行い、学習用CSVファイルを効率よく生成できます。
DCON2026に出場するプロダクトの開発のために作成しました。

災害救助AI（例：Victim / Rescuer / Environment の分類）をはじめとした、**音声データを用いた研究・開発用途**を想定して設計されています。

---

## 特徴

- **GUIによる直感的な音声アノテーション**
- **分類タスク・回帰タスクの両方に対応**
- **柔軟なラベル設計（起動時に自由に定義可能）**
- **CSV形式でのデータセット出力**
- **ファイル名整理（リネーム）機能付き**

Python標準ライブラリの `tkinter` と音声再生用の `pygame` を使用しています。

---

## 想定ユースケース

- 災害救助音（要救助者・救助隊・環境音）の分類
- 人がいそう度（0.0〜1.0）のスコア付与
- 音声イベント検出・音響シーン分類
- 研究・卒業研究・ハッカソン等でのデータ作成

---

## 主な機能

### 1. 柔軟なラベル設定
- 起動時に任意のクラス名を定義可能  
  （例：`victim`, `rescuer`, `environment`）
- タスクに応じて自由に変更可能

### 2. 数値スコア（回帰ラベル）対応
- クラス分類だけでなく  
  **0.0〜1.0 などの数値スコア入力**にも対応
- 「人がいそう度」などの連続値ラベル作成が可能

### 3. オーディオプレイヤー
- 再生 / 一時停止 / 停止
- 音量調整スライダー
- ファイル切り替え時の自動停止

### 4. 効率的なアノテーション操作
- ファイル一覧からワンクリックでラベル付与
- ラベル付与後、自動で次のファイルへ移動
- 連続作業に最適化されたUI設計

### 5. CSVエクスポート
- 学習用データセット定義ファイルを生成
- フォーマット例：

filename,label
dog_bark_01.wav,environment
walking_05.wav,rescuer
voice_03.wav,victim
